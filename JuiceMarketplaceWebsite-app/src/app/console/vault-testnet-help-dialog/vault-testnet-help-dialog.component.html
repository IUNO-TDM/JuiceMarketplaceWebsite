Informationen zur Währung IUNO

Die Währung in unserem Demonstrator nennt sich IUNO. Hinter den IUNOs steckt technisch das Bitcoin Testnet Diese Variante von Bitcoin ist zum Testen und zum Entwickeln mit der Bitcoin-Technik gedacht. Einen realen Wert, den man beispielsweise in Euros ausdrücken könnte, gibt es nicht. Jedoch kann man damit Cocktails an unseren Saftmischmaschinen bezahlen.

Für dich als Technologiedatenhersteller speichern wir deine IUNOs, die du mit Lizenzverkäufen erhalten hast. Wenn du willst, kannst du dir dein Guthaben gerne an deine Bitcoin-Wallet schicken. Dazu benötigst du eine Bitcoin-Wallet, die mit dem Bitcoin-Testnet zusammenarbeitet.

Folgende Apps können wir dazu empfehlen:

Windows, Linux, Mac: Bitcoin Core
iOS: Copay (Nur im Testnet empfehlenswert)
Android: Bitcoin Wallet for Testnet
